# Gender-Health-Care-System

Gender Health Care System is an inclusive healthcare system dedicated to supporting well-being of people of all genders.

## Modules
- Multi-role user management
- Online medical consultation
- Testing services
- Women's health monitoring
- VNPay integrated payment

## Technologies
- Spring Boot, Java 21
- React, Ant Design
- SQL Server
- JWT, OAuth2
- Firebase, Cloudinary

## About
Gender Health Care System is an inclusive healthcare system providing personalized medical services for all genders.
